# Mini Projetos Python

Este repositório contém diversos pequenos projetos e exercícios em Python que fiz durante meus estudos.

## Projetos incluídos:
- Calculadora de IMC
- Simulação de Empréstimo
- Calculadora de Alistamento Militar
- Verificador de Número Maior
- Calculadora de Multiplicação
- Separação de Idade em Classes
- Calculadora para Área de Pintura
- Calculadora de Recuperação

Cada arquivo é independente e pode ser executado diretamente.
