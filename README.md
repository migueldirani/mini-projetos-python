# Mini Projetos Python

Este repositório contém 10 exercícios em Python que fiz durante meus estudos.

## Exercícios incluídos

- 1_calculadora_imc – Calculadora de IMC
- 2_alistamento_militar – Aviso de alistamento militar
- 3_simulador_emprestimo – Simulador de empréstimo bancário
- 4_maior_numero – Verificador de número maior
- 5_tabuada – Mostra a tabuada de um número
- 6_classificacao_idade – Classifica idade em categorias
- 7_calculadora_imc_detalhada – IMC com classificação detalhada
- 8_area_pintura – Calcula área e tinta necessária
- 9_jogo_adivinhacao – Pequeno jogo de adivinhação (ou outro exercício que escolher)
- 10_calculadora_recuperacao – Calcula se o aluno está aprovado, recuperação ou reprovado

## Como usar

1. Clone ou baixe este repositório.  
2. Abra o projeto no PyCharm ou outro editor de sua preferência.  
3. Execute o arquivo `.py` correspondente ao exercício que deseja rodar.

**Obs:** A pasta `.idea/` foi ignorada, então não será enviada para o GitHub.
